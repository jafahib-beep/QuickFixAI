QuickFix – Full Feature Specification

Purpose
QuickFix is a clean, minimal, fast mobile app for short 30–60 second community videos showing how to fix everyday problems (home, car, electronics, tools, cleaning, etc.). Users can search, watch, save, and upload quick troubleshooting videos. The UI is simple, professional, and user-friendly.

Core Features
1. User Registration & Login

Email + password registration

Social login (Google & Apple)

Required fields:

Display name

Email

Password

Secure token/session system (JWT or similar)

“Forgot password” and password reset flow

Redirect to onboarding after sign-up

Login UI Copy

Title: “Welcome back”

Buttons: “Log in”, “Continue with Google”, “Continue with Apple”

Link: “Forgot password?”
